# Slowly返信プロンプト生成ツール

`test.html` をブラウザで開くだけで使える、スタンドアロンのフロントエンドアプリです。

## できること
- 文通相手の出身国・地域、返信言語、丁寧さ、関係段階、ラリー回数、過去文通の要約を入力
- 最新の手紙本文を貼り付けて、カスタムGPT向けの指示プロンプトを生成
- 生成結果のコピー、`.txt` 保存
- 入力内容を `localStorage` に自動保存

## 使い方
1. `test.html` をブラウザで開く
2. 各項目を入力して「プロンプトを生成」をクリック
3. 必要に応じて「生成結果をコピー」または「.txtで保存」を実行
